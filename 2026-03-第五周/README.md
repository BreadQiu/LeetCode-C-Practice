# 第五周（2026.03.30-2026.03.31）

## 本周概览
本周主要练习了一些数学、字符串和数组的简单题，巩固了循环、取模、位运算等基础操作。题目整体难度较低，适合保持手感。

## 学习重点
- 数学题：各位相加、丑数、3的幂、2的幂、最小偶倍数
- 字符串转换：转换成小写字母
- 数组操作：重新排列数组、数组异或操作、好数对、统计好三元组
- 简单二叉树判断
