# 第七周（2026.04.08-2026.04.13）

## 本周概览
本周是大一下学期的关键阶段——分流志愿填报已经尘埃落定，我最终将**高分子材料与工程**填为第一志愿，**无机非金属材料工程**填为第二志愿。这两周一边准备分流，一边继续刷题保持手感。同时，我开始全力准备4月27日至30日的计算机学院转专业申请材料的最后打磨。

本周主要练习了**二分查找**（在排序数组中的各种变体）和**滑动窗口**（无重复子数组）专题，也做了一些数学统计类题目。总体来看，二分查找的手动实现仍然是我的弱项，在边界条件的处理上容易出错。

本周的练习题主要来自**LeetCode第179场双周赛**和日常练习：
- 4月8日：1695. 删除子数组的最大得分（滑动窗口，周赛题）
- 4月9日：34. 在排序数组中查找元素的第一个和最后一个位置（二分边界）
- 4月10日：35. 搜索插入位置（二分边界）
- 4月11日：704. 二分查找（基础二分）
- 4月13日：744. 寻找比目标字母大的最小字母（二分边界）、2529. 正整数和负整数的最大计数（二分统计）
