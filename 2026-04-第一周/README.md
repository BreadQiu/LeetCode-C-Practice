# 第六周（2026.04.01-2026.04.07）

## 本周概览
本周重点练习了滑动窗口（水果成篮、尽可能使字符串相等、删掉一个元素以后全为1的最长子数组）、双指针、矩阵转置和前缀和等题目。难度有所提升，部分题目在边界处理和代码简洁性上仍有改进空间。

## 学习重点
- 滑动窗口：水果成篮、尽可能使字符串相等
- 双指针/前缀和：分割字符串的最大得分、删掉一个元素以后全为1的最长子数组
- 数学与位运算：2的幂
- 矩阵操作：转置矩阵
- 数组平衡问题：使数组平衡的最少移除数目
